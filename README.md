
# LinkSphere -Saas Platform

**LinkSphere** is a **SaaS (Software-as-a-Service) platform** where creators, influencers, and businesses can create a single beautiful page to share ALL their links. Instead of struggling with "link in bio" limits on Instagram, TikTok, or Twitter, users share just ONE LinkSphere profile containing all their important links with advanced analytics and audience insights.

**Think of it as:** Linktree +Analytics, all in one platform.


## 🎯 The Problem We Solve

### The Creator's Struggle

| Problem | Example |
|---------|---------|
| **Instagram allows only 1 link** | "LinkSphere" - but you have 10 links to share |
| **No analytics** | You don't know who clicked what or from where |
| **Ugly generic pages** | Every Linktree page looks the same |
| **No audience insights** | You don't know your audience demographics |

### Our Solution

| Problem | LinkSphere Solution |
|---------|---------------------|
| 1 link limit | ONE link that contains ALL links |
| No analytics | Full analytics dashboard with charts |
| Ugly pages | Fully customizable themes & colors |
| No insights | Audience intelligence with demographics |

## Features

### 🔐 Authentication & User Management
- ✅ User registration with email/password
- ✅ JWT-based authentication
- ✅ Password hashing with bcrypt
- ✅ Refresh token mechanism
- ✅ Role-based access (User / Admin)
- ✅ Protected routes & API endpoints

### 👤 Profile Management
- ✅ Custom username (linksphere.com/@username)
- ✅ Profile picture upload (Cloudinary)
- ✅ Bio (140 characters with emoji support)
- ✅ Social links (Instagram, YouTube, Twitter, TikTok)
- ✅ Email & account settings

### 🎨 Customization (Full Control)
- ✅ **10+ Premium Themes** (Obsidian, Aurora, Sunset, etc.)
- ✅ **Custom Button Colors** - Any hex color
- ✅ **Custom Text Colors** - Heading & body text
- ✅ **Font Selection** - 10+ Google Fonts
- ✅ **Button Styles** - Pill, Rounded, Square, Soft, Neon, Glass
- ✅ **Background Images** - Upload custom background (Pro)
- ✅ **Live Preview** - See changes in real-time

### 🔗 Link Management (CRUD)
- ✅ Create unlimited links
- ✅ Edit link title, URL, icon
- ✅ Delete links
- ✅ Drag & drop reordering
- ✅ Thumbnail images for links
- ✅ Custom icons & emojis
- ✅ Links open in new tab

### 📊 Analytics Dashboard
- ✅ **Profile Views** - Total & unique visitors
- ✅ **Link Clicks** - Per link performance
- ✅ **Click-Through Rate (CTR)** - Engagement metrics
- ✅ **Device Analytics** - Mobile vs Desktop vs Tablet
- ✅ **Location Tracking** - Visitors by country/city
- ✅ **Browser Statistics** - Chrome, Safari, Firefox, etc.
- ✅ **Time Analytics** - Peak hours & best days
- ✅ **Trend Charts** - Line, bar, pie charts (Recharts)
- ✅ **Date Filtering** - 7d, 30d, 90d, custom range

### 👥 Audience Intelligence
- ✅ **Demographics** - Age, gender, interests (coming from IP)
- ✅ **Geographic Distribution** - Heat map of visitors
- ✅ **Device Breakdown** - iPhone, Android, Desktop
- ✅ **Returning vs New Visitors** - Retention metrics
- ✅ **Behavior Patterns** - Scroll depth, time on page
- ✅ **Popular Links Ranking** - Most clicked first

### 📱 Public Profile Page
- ✅ Beautiful, responsive design
- ✅ Applies user's theme & colors
- ✅ Displays profile picture, name, bio
- ✅ Shows social media icons
- ✅ Lists all links in custom order
- ✅ Click tracking built-in
- ✅ View counting
- ✅ Newsletter signup (Pro feature)
- ✅ QR code generation (Pro feature)
- ✅ Share button - Copy profile link

### 👑 Admin Panel
- ✅ **User Management** - View all users, edit, suspend
- ✅ **Platform Analytics** - Total users, views, clicks
- ✅ **Revenue Tracking** - Pro subscribers, MRR
- ✅ **System Settings** - Configure platform
- ✅ **Role Management** - Promote users to admin

### 🖼️ Image Upload
- ✅ Profile picture upload (Cloudinary CDN)
- ✅ Cover background image (Pro users)
- ✅ Drag & drop support
- ✅ Instant preview
- ✅ File validation (images only)
- ✅ Size limit (5MB)
- ✅ Format support (JPG, PNG, GIF, WebP)

### 📱 Responsive Design
- ✅ Mobile-first approach
- ✅ Tablet optimized
- ✅ Desktop ready
- ✅ All devices supported

## Run Locally

Clone the project

```bash
  git https://github.com/boominathan07/-LinkSphere.git
```

Go to the project directory

```bash
  cd LinkSphere
```

Backend setup

```bash
cd server
npm install
# Edit .env with your MongoDB URI and Cloudinary credentials
npm run dev
```
Frontend setup (open new terminal) 

```bash
cd server
npm install
# Edit .env
npm run dev
```

Start the server

```bash
  npm run start
```


## Deployment

```bash
  npm run build
```
